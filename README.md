## Installation 安装

```bash
cd hexo
git clone https://github.com/DaxiLi/aria-cards-hexo.git themes/cards

vi _config.yml
-theme: landscape
+theme: cards

hexo s --debug
```

你也可以参照主题文档详细了解相关配置。
